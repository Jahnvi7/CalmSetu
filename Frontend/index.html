This single‑file prototype demonstrates the CalmSetu UI: Landing → Auth → Dashboard with Profile & Settings. No backend yet; perfect for hackathon demos.

What’s inside

  • Responsive layout with **Landing, About, Login, Register**
  • Fake auth → switches to **Dashboard** (no server yet)
  • Dashboard with **Home, Profile, Settings** tabs
  • Dark mode toggle, clean cards, soft gradients

About CalmSetu

CalmSetu is a Gen‑AI concept for youth mental wellness. This prototype focuses on UI/UX only. You can attach a Python backend (Flask/FastAPI) and SQL database later.
